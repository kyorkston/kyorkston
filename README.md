## :hammer: Develop project

1.  Start development server:

    ```sh
      gatsby develop
    ```

    Runs on `http://localhost:8000` and the graphiql interfact runs on `http://localhost:8000/___graphql`

2.  Build project:

    ```sh
      gastby build
    ```
